# jstility.github.io

Source code for the [JStility](https://jstility.github.io) homepage.

Built with [Next.js](https://nextjs.org) and [Nextra](https://nextra.site).

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The static output is generated in the `out/` directory.
